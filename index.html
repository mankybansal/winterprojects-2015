CRYPTOGRAPHY
DESCRIPTION
Objective: To learn the Python language and work on a project.

In today's digitised world, most content is in the form of images and video. Information security is one of the most growing priorities today and secure encryption algorithms are required to protect sensitive content from unauthorised access. The aim of this winter project is to learn about the basic concepts of cryptography, understand how images can be encrypted and what different loopholes need to be addressed while encrypting images and finally to develop a new algorithm to encrypt images.

The students will learn the language by the 31st of December. Basics like expressions and Data types, functions, loops, conditions, files, lists, etc will be covered. When that is done, they will be given a set of questions to assess themselves with. They will then work on a project over the next ten days, which will be told to them within the next two weeks.
PREREQUISITES
Knowledge of basic MATLAB functions, understanding of pixel-pixel representation of RGB images(tri-plane structure)

Duration: 4 Weeks

COMPETITIVE CODING + WCS
DESCRIPTION
Objective(s):
A) To enhance one's competitive coding skills
B) Ensure positive competition among the WC members
C) Select members who can help with the WCS

Points will be awarded to each one individually based on their performance.
1. We will give them the FIRST question.
2.Person 'A', winning the first round, will set the next question, which will be verified by us.
3. The winner of the second round will set the next question and so on.
4. A 24 or 48 (to be decided) hour gap will be given to attempt/solve each question. Questions will be uploaded at night.
5. Test cases will be verified by us and we will help them with the test cases in case the question is tough.
6. Points will be awarded to them in accordance with the last WCS scoring system.
7. Partial test cases will hold some points as well (so that they don't get demotivated).
8. Few members with high cumulative scores will help with the WCS,this year.

Also, we plan to prep them with a few topics like STL, Vectors and Time Complexities.

Members should have attended AT LEAST 2 code-meet sessions.
